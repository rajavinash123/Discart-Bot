

# 🎓 Placement Support Discord Bot

A powerful Discord bot designed for placement preparation, student support, and career guidance communities.

## 🚀 Features

- 📢 Placement updates & announcements
- 🧠 AI-based student support
- 📄 Resume review system
- 🎯 Interview preparation channels
- 🏢 Company-wise discussion rooms
- 📅 Placement drive reminders
-

## 🛠️ Tech Stack

- Node.js
- Discord.js
- MongoDB
- Express.js
